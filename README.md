<h1>Spotify App</h1>

A way to view your favorite artists, and tracks via Spotify API.

<br>

<p>To setup locally:</p>
<ul>
  <li>Clone repo into empty directory</li>
  <li>Run `npm i`</li>
  <li>Run `npm run start`</li>
  <li>Navigate to http://localhost:3000 in your browser</li>
</ul>
